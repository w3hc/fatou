# Fatou

Fatou is a Nest.js-based API that interacts with several different LLM services.

## Installation

```bash
pnpm i
```

## Running the app

```bash
# development
pnpm start

# watch mode
pnpm start:dev

# production mode
pnpm start:prod
```

## Test

```bash
# unit tests
pnpm test

# e2e tests
pnpm test:e2e

# test coverage
pnpm test:cov
```
